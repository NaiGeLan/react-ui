## Usage

```
pnpm i
pnpm run dev
pnpm run build
```

## Options

- 样式
  - 引号：单引号
  - 缩进：两空格
  - Trailing Commas：es5

## Development

```bash
# install dependencies
$ pnpm install

# develop library by docs demo
$ pnpm start

# build library source code
$ pnpm run build

# build library source code in watch mode
$ pnpm run build:watch

# build docs
$ pnpm run docs:build

# check your project for potential problems
$ pnpm run doctor
```
